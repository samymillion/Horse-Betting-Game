void GameWindow::onBetEntered ( )
privateslot

A function that once the user has entered bet sets the bet amount and goes onto the next screen to confirm if it is placed succesfully or if there is an error.

After the user enters a bet and presses the enter key, it checks to see if the bet is valid and within the bounds of the money pool. If it is valid, the user may start the race. Otherwise, they are shown a warning message. It prohibits the user from entering a new bet while a race is in progress.
